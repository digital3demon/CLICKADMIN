dental-lab-crm — переносная папка (Windows)
===========================================

На компьютере уже установлен Node.js LTS (https://nodejs.org/, галка Add to PATH).

Скопируйте всю эту папку куда угодно — с диска, с флешки, без «установки» в систему.

Дважды нажмите «Запуск.bat» или «START.bat» (одинаковое содержимое; START — если
кириллическое имя не запускается).

--- Если в папке prisma есть файл .PORTABLE_FULL_DB (сборка с полной базой) ---

В комплекте уже лежит ваша база: prisma\dev.db. Скрипт НЕ делает prisma db push и
НЕ затирает данные seed — сразу запускается сервер. Первый запуск = открыть браузер.

--- Если маркера .PORTABLE_FULL_DB нет (пустая переносимая сборка) ---

Скрипт синхронизирует SQLite со schema.prisma (prisma db push), затем подставляет
демо-данные (клиника, врачи, наряды). Файл базы: prisma\dev.db

------------------------------------------------------------------------

Когда в чёрном окне появится адрес — откройте в браузере:

    http://localhost:3000

Остановить программу: закройте окно или нажмите Ctrl+C в нём.

Если страница «голая» или не нажимается: откройте http://127.0.0.1:3000
или переупакуйте папку свежей сборкой npm run package:windows (исправление путей).

Если раньше уже запускали старую сборку и «пусто» или ошибки — удалите prisma\dev.db
и файл .env в этой папке, затем снова Запуск.bat (создастся новый .env из примера).
Для сборки С полной базой снова запустите package:windows на машине, где есть
актуальный prisma\dev.db.


Сборка у разработчика (в корне репозитория)
-------------------------------------------

  npm run package:windows

Копируется prisma\dev.db из проекта, если файл есть. Иначе в архиве будет пустая
схема + seed при первом запуске.

Указать другой файл SQLite:

  powershell -NoProfile -ExecutionPolicy Bypass -File scripts/package-standalone-windows.ps1 -SqliteSourcePath "C:\data\my-crm.db"

Собрать без вшитой базы (как раньше: только db push + seed на первом запуске):

  powershell -NoProfile -ExecutionPolicy Bypass -File scripts/package-standalone-windows.ps1 -SkipBundledDatabase

Архив: dist\dental-lab-crm-portable.zip
